
## 👋 소개
회원가입, 로그인 기능이 있는 투두앱 백엔드 서버 제작 프로젝트

## 🛠️ 개발 환경
- 언어: Java
- 버전: JDK 17
- 개발 환경: IntelliJ

## 🦻🏻 주요 구현 사항
- JWT를 이용한 회원가입, 로그인
- 로그인 한 사용자에 한하여 일정과 댓글을 작성하고 수정 가능
- 만료되지 않은 유효 토큰인 경우에만 일정과 댓글 생성 가능
